Fantasy Festival Vue project

The Fantasy Festival Vue project involved creating a single-page application (SPA) to differentiate the quest from the main festival website. Starting with a sitemap, wireframes, and high-fidelity mockups, we designed the site to be simple, visually appealing, and highly functional. It includes responsive design, interactive galleries using anime.js, and small animations, while trust indicators such as SSL certification, an interactive map, and mobile friendliness enhance credibility. Built using Vue.js for dynamic components and deployed through Firebase, the website prioritizes ease of use, sustainability, and user security. Face-to-face collaboration allowed for efficient problem-solving, minimizing the need for GitHub.
